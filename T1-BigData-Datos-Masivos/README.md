# 📊 Big Data y Datos Masivos en las Organizaciones

**Trimestre 1 · Marzo – Mayo 2026**  
**Universidad DaVinci · Maestría en Ciencia de Datos**

---

## 📋 Descripción

Asignatura de fundamentos del Big Data en el contexto organizacional. Cubre las 7 Vs del Big Data, arquitecturas de procesamiento distribuido, ecosistema Hadoop, cloud computing, sistemas de almacenamiento masivo (DAS, NAS, SAN, SDS), virtualización de datos, Open Data, data mining y bases de datos NoSQL con MongoDB.
