# 📊 UD11-12 — Las 7 Vs del Big Data y Fuentes de Datos

**Trimestre 1 · S8 (May 2026)**

---

## 1. Las 7 Vs del Big Data

| V | Definición | Ejemplo Movet |
|---|---|---|
| **Volumen** | Tamaño masivo | Millones de registros de pacientes |
| **Velocidad** | Rapidez de generación | Alertas de salud animal en tiempo real |
| **Variedad** | Formatos heterogéneos | Texto, imágenes, datos estructurados |
| **Veracidad** | Calidad y confiabilidad | Diagnósticos incorrectos = decisiones erróneas |
| **Valor** | Utilidad para el negocio | Predicción de enfermedades estacionales |
| **Variabilidad** | Inconsistencia en el tiempo | Estacionalidad en tipos de consultas |
| **Visualización** | Capacidad de presentar los datos | Dashboards Power BI para gerencia |

---

## 2. Fuentes de datos

| Tipo | Descripción | Ejemplo |
|---|---|---|
| **Estructurados** | Formato fijo, tablas, SQL | BD Odoo ERP de Movet |
| **Semiestructurados** | Formato flexible, JSON/XML | Logs de aplicaciones, APIs |
| **No estructurados** | Sin formato predefinido | Imágenes médicas, audio, texto libre |
| **Tiempo real (streaming)** | Flujo continuo de eventos | Sensores IoT, redes sociales |
| **Open Data** | Datos públicos disponibles | DANE, Banco Mundial, datos.gov.co |
