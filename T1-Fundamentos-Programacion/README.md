# 📘 Fundamentos de Programación — JavaScript

**Trimestre 1 · Marzo – Mayo 2026**  
**Universidad DaVinci · Maestría en Ciencia de Datos**

---

## 📋 Descripción

Asignatura de introducción a la programación con JavaScript. Cubre desde los conceptos más básicos (variables, tipos de datos, operadores) hasta temas avanzados como programación orientada a objetos, manipulación del DOM/BOM, consumo de APIs con AJAX y uso de frameworks como jQuery.
